# epl_temp/graphs

graph files are labeled accordingly:

graph_epl[type]_temp_[age group]

type is defined by: <br />
P = EPL for permanent workers <br />
T = EPL for temporary workers <br />
G = gap (P - T)

age group is defined by: <br />
A = All ages <br />
P = Prime age (25-54) <br />
Y = Young age (15-24)
